üéØTutorial 1: Sales Funnel Design

Goal: Create a multi-step customer journey from awareness to purchase

Step 1: Define Target Audience & Map Customer Journey

  • Create buyer persona (demographics, pain points, goals, objections)
  • Map current buying journey: How do they discover you? What questions do they have?
  • Identify touchpoints: social media, ads, email, website, sales call

Step 2: Choose Your Funnel Type

  • Lead Generation Funnel: Free offer ‚Üí Email capture ‚Üí Nurture sequence ‚Üí Sales call
  • Product Launch Funnel: Pre-launch hype ‚Üí Early bird offer ‚Üí Main launch ‚Üí Cart close
  • Webinar Funnel: Registration ‚Üí Reminder emails ‚Üí Live webinar ‚Üí Replay sequence
  • Tripwire Funnel: Low-cost offer ($7-$47) ‚Üí Upsell ‚Üí High-ticket backend
  • Application Funnel: Info page ‚Üí Application form ‚Üí Qualifying call ‚Üí Enrollment

Step 3: Map Out Funnel Stages (AIDA Framework)

  • Awareness: Ads, social posts, blog ‚Üí Landing page with hook
  • Interest: Educational content, case studies, testimonials
  • Desire: Show transformation, benefits, urgency/scarcity
  • Action: Clear CTA, easy checkout, follow-up sequence

Step 4: Create Wireframes for Each Page

  • Use Figma, Canva, or pen and paper to sketch layouts
  • Page 1: Landing page with headline, benefits, CTA, social proof
  • Page 2: Thank you/delivery page with next steps
  • Page 3: Sales page with offer details, pricing, FAQ
  • Page 4: Checkout page (simple, minimal fields)
  • Page 5: Order confirmation with upsell/cross-sell

Step 5: Write Compelling Copy for Each Stage

  • Headlines: State clear benefit or solve pain point (e.g., "Get 50 Qualified Leads in 30 Days")
  • Subheadlines: Expand on headline, overcome objection
  • Body Copy: Use PAS formula (Problem, Agitate, Solution)
  • Bullet Points: Specific benefits, not features (transformation-focused)
  • CTAs: Action-oriented, benefit-driven (e.g., "Get My Free Guide" not "Submit")

Step 6: Design Pages in Your Platform

  • Choose tool: ClickFunnels, Unbounce, Leadpages, GoHighLevel, or custom HTML
  • Use templates as starting point, customize to brand
  • Keep design clean: plenty of white space, one clear goal per page
  • Use high-quality images and videos (stock or client-provided)
  • Add trust signals: testimonials, logos, security badges, guarantees

Step 7: Set Up Tracking & Analytics

  • Install Google Analytics 4 and Facebook Pixel on all pages
  • Set up conversion goals for each step (page view, form submit, purchase)
  • Track key metrics: traffic source, bounce rate, conversion rate, average order value
  • Use Hotjar or Microsoft Clarity for heatmaps and session recordings

Step 8: Test and Optimize

  • Before Launch: Test all links, forms, payment processing, mobile view
  • Week 1-2: Monitor funnel flow, identify drop-off points
  • Week 3-4: A/B test headlines, CTAs, images
  • Ongoing: Run split tests every 2-4 weeks, improve conversion rate by 5-10%
üí° Pro Tip: Document your funnel in a visual flowchart (use Miro or Lucidchart) to share with clients and show your strategic thinking

üé®Tutorial 2: Landing Page Creation

Goal: Create a high-converting page with one specific conversion goal

Step 1: Define Single Conversion Goal

  • Email signup? Product purchase? Webinar registration? Call booking?
  • Remove all other distractions and navigation (no menu, no external links)
  • Every element should guide visitor toward ONE action

Step 2: Write Benefit-Driven Headline

  • Use formula: [Desired Outcome] without [Pain Point] in [Timeframe]
  • Example: "Build Your First Sales Funnel Without Coding in 7 Days"
  • Make it specific, quantifiable, and desirable
  • Test 5-10 variations before choosing winner

Step 3: Create Compelling Hero Section

  • Above the fold (no scrolling needed):
  • Headline (largest text, 36-48px on desktop)
  • Subheadline explaining benefit or overcoming objection
  • Hero image or video (showing product/result)
  • Primary CTA button (contrasting color, 2-3 words + benefit)
  • Trust indicators: "As seen on", client logos, rating stars

Step 4: Add Social Proof Section

  • Testimonials: 3-5 with photos, names, company, specific results
  • Case Studies: Before/after metrics ("Increased revenue by 143%")
  • Client Logos: Recognizable brands if available
  • Stats: "Join 10,000+ happy customers"
  • Place social proof near CTA buttons for maximum impact

Step 5: Design Clear CTA Buttons

  • Include 2-3 CTAs: above fold, middle of page, bottom
  • Use action verbs: "Get", "Start", "Claim", "Download", "Reserve"
  • Add benefit: "Get My Free Template" > "Submit"
  • Make buttons large (min 44x44px for mobile), contrasting color
  • Add micro-copy below: "No credit card required" or "Instant access"

Step 6: Mobile Optimization Checklist

  • ‚úÖ Text readable without zooming (min 16px font size)
  • ‚úÖ Buttons easily tappable (min 44px height)
  • ‚úÖ Forms have large input fields and minimal fields (3-5 max)
  • ‚úÖ Images load fast (compressed, WebP format)
  • ‚úÖ Page loads in under 3 seconds on mobile
  • ‚úÖ Test on iPhone and Android devices

Step 7: A/B Test These Elements

  • Headline variations (benefit vs curiosity vs direct)
  • Hero image vs video vs illustration
  • CTA button color (red, green, orange perform well)
  • CTA copy ("Get Started Free" vs "Start My Free Trial")
  • Form length (2 fields vs 5 fields)
  • Social proof placement and format
üí° Conversion Rate Benchmarks: 1-3% is average, 5-10% is good, 10%+ is excellent. Focus on improving incrementally.

üìßTutorial 3: Email Sequence Design

Goal: Create automated email flows that nurture leads and drive sales

Step 1: Map Subscriber Journey (Days 1-30)

  • Day 1: Welcome email + deliver lead magnet
  • Days 2-7: Education and value (no selling)
  • Days 8-14: Soft pitch with case studies
  • Days 15-21: Direct offer with urgency
  • Days 22-30: Re-engagement or segmentation

Step 2: Write Email Sequence Outline

  • Welcome Email: Set expectations, deliver promised content, tease what's coming
  • Value Emails (3-5): Tips, how-tos, case studies, stories (80% value, 20% promo)
  • Sales Emails (3-5): Introduce offer, overcome objections, add urgency/scarcity
  • Retention Emails: Check-ins, surveys, re-engagement campaigns

Step 3: Craft High-Open-Rate Subject Lines

  • Curiosity: "The one mistake costing you 50% of your leads"
  • Benefit: "How to double your conversion rate in 48 hours"
  • Personalization: "[Name], I noticed you haven't [action] yet"
  • Urgency: "Last chance: Doors close in 6 hours"
  • Keep under 50 characters for mobile preview
  • Avoid spam words: "Free", "Buy now", "Act now", "!!!"

Step 4: Write Emails with Clear CTAs

  • Opening: Personal greeting, reference previous email or action
  • Body: Tell story or share insight (keep scannable with short paragraphs)
  • CTA: One clear action per email (read blog, watch video, buy product)
  • P.S.: Reinforce CTA or add secondary offer
  • Write like you're emailing a friend (conversational, not corporate)

Step 5: Set Up Automation in Email Platform

  • Use tools: Mailchimp, ConvertKit, ActiveCampaign, Klaviyo
  • Create automation workflow with triggers (e.g., "when someone downloads X")
  • Set delays between emails (1-3 days for nurture, hours for urgency)
  • Add conditional logic: "If clicked link A, send email B"
  • Tag subscribers based on behavior for segmentation

Step 6: Monitor Key Metrics

  • Open Rate: 15-25% is average, 30%+ is excellent
  • Click-Through Rate: 2-5% is average, 10%+ is excellent
  • Conversion Rate: Track sales from email campaigns
  • Unsubscribe Rate: Under 0.5% per email is healthy
  • Analyze which emails perform best and replicate that style
üí° Best Send Times: Tuesday-Thursday, 10am or 2pm in subscriber's timezone. Test for your specific audience!

üõíTutorial 6: E-commerce Product Funnel

Goal: Maximize revenue per customer with upsells, cart abandonment recovery, and order bumps

Step 1: Product Page Optimization

  • High-quality images: 5-7 images showing product from multiple angles, lifestyle shots
  • Video demonstration: 30-60 second video showing product in use
  • Benefit-focused bullets: Not just features - show transformation
  • Social proof: Display 5-10 reviews with star ratings, photos from customers
  • Urgency elements: "Only 3 left in stock" or "Sale ends in 6 hours"
  • Clear CTA: Large "Add to Cart" button in contrasting color

Step 2: Cart Page Upsells & Order Bumps

  • Order Bump: Checkbox offer on cart page: "Add [complementary product] for only $X more"
  • Quantity Breaks: "Buy 2, Get 15% Off" or "Buy 3, Get 25% Off"
  • Free Shipping Threshold: "Add $15 more for free shipping"
  • Product Bundles: "Customers also bought these together" (save X%)
  • Keep cart page simple - don't overwhelm with too many options

Step 3: Checkout Page Optimization

  • Single-page checkout (don't make them click through multiple pages)
  • Guest checkout option (don't force account creation)
  • Multiple payment options: Credit card, PayPal, Apple Pay, Google Pay
  • Trust badges: SSL secure, money-back guarantee, secure payment icons
  • Progress indicator if multi-step: "Step 2 of 3"
  • Auto-fill address fields using Google API
  • Mobile-optimized form fields (large tap targets)

Step 4: Post-Purchase Upsell (One-Click Upsell)

  • Thank you page offer: "Wait! Special offer just for you"
  • One-click purchase: No re-entering payment info
  • Offer types: Upgraded version, complementary product, subscription
  • Headline: "Since you bought X, you'll love Y (50% off today only)"
  • Countdown timer: "Claim this offer in the next 10 minutes"
  • Success rate: 10-30% of customers take post-purchase upsell

Step 5: Cart Abandonment Email Sequence

  • Email 1 (1 hour later): "You left something behind" + show cart items
  • Email 2 (24 hours): "Still thinking it over? Here are reviews" + testimonials
  • Email 3 (48 hours): "Last chance: 10% discount if you complete order today"
  • Include direct "Complete Purchase" button that takes them to checkout
  • Show product image and name in email
  • Recovery rate: 10-15% of abandoned carts can be recovered

Step 6: Post-Purchase Email Flow

  • Order confirmation (immediate): Receipt, shipping timeline, tracking info
  • Shipping notification: "Your order has shipped" + tracking link
  • Delivery day email: "Your package arrives today! Here's how to use it"
  • Follow-up (7 days): "How's your [product]? Leave a review, get 15% off next order"
  • Replenishment (30-60 days): For consumables, "Time to restock?"
  • Cross-sell (90 days): Recommend related products based on purchase
üí° E-commerce Benchmark: Average cart abandonment rate is 70%. Recovering even 10% with email sequences can dramatically increase revenue.

Goal: Systematically improve conversion rates through split testing

Step 1: What to Test (Priority Order)

  • Headline: Biggest impact on conversion (test first)

Step 2: Setting Up A/B Tests

  • Native tools: Google Optimize (free), Unbounce, ClickFunnels built-in
  • Third-party tools: VWO, Optimizely, Convert.com
  • Split traffic: 50/50 between Control (A) and Variant (B)
  • Sample size: Need minimum 100 conversions per variant for statistical significance
  • Test duration: Run for at least 7-14 days (capture full week cycle)
  • Don't stop tests early - let them run full course

Step 3: Statistical Significance

  • Confidence level: Aim for 95% confidence minimum
  • Calculator tools: Use A/B test calculators (ABTestGuide.com, Optimizely's calculator)
  • Example: Control: 100 visitors, 5 conversions (5%) | Variant: 100 visitors, 8 conversions (8%)
  • Is 8% > 5% significant? Run through calculator to verify
  • Avoid: Calling winners too early, testing too many things at once

Step 4: Headline Testing Framework

  • Control: Your current headline
  • Variant 1: Benefit-focused: "Get [Desired Outcome] in [Timeframe]"
  • Variant 2: Pain-focused: "Stop [Pain Point] Forever"
  • Variant 3: Question format: "Want to [Desired Result]?"
  • Variant 4: Social proof: "Join 10,000+ Who've Achieved [Result]"

  • Create testing log: Date, element tested, results, winner

  • Share results with clients in monthly reports
  • Never stop testing - optimization is ongoing

üß†Funnel Psychology & Persuasion Principles

Understanding human psychology is the secret to high-converting funnels. Use these proven principles ethically to influence buying decisions.

1. Scarcity Principle

What it is: People value things more when they're limited or running out.

  • Countdown timers: "Offer expires in 3 hours 42 minutes"
  • Limited quantity: "Only 5 spots remaining"
  • Seasonal availability: "This course only opens twice a year"
  • Warning: Must be genuine - fake scarcity damages trust
  • Example: "Early bird pricing ends Friday at midnight"

2. Urgency Principle

What it is: Creating a reason to act NOW instead of later.

  • Time-limited bonuses: "Order today and get [bonus] free"
  • Price increases: "Price goes up $200 on Sunday"
  • Event-based: "Register by Friday to attend live workshop"
  • Psychology: Fear of missing out (FOMO) triggers immediate action
  • Example: "Last chance: Sale ends in 24 hours"

3. Social Proof

What it is: People follow what others are doing, especially similar people.

  • Testimonials: Real customer stories with photos, names, results
  • Numbers: "Join 50,000+ satisfied customers"
  • Case studies: Detailed before/after transformations
  • Expert endorsements: Industry authorities recommending you
  • Media mentions: "As seen in Forbes, TechCrunch, Inc."
  • User activity: "127 people viewing this page right now"

4. Reciprocity Principle

What it is: When you give value first, people feel compelled to give back.

  • Free lead magnets: Give valuable content before asking for sale
  • Free trials: Let them experience product value first
  • Bonuses: Add extra value to purchase
  • Free consultations: Give advice before pitching service
  • Example: "Download this free template, then book a call"

5. Authority Principle

What it is: People trust and follow recognized experts.

  • Credentials: Certifications, degrees, years of experience
  • Awards: "Winner of 2025 Marketing Innovation Award"
  • Media appearances: Podcast interviews, speaking engagements
  • Published content: Books, articles in major publications
  • Results: "$10M+ generated for clients" establishes expertise

6. Commitment & Consistency

What it is: People want to act consistently with their previous commitments.

  • Micro-commitments: Start with small asks (free download) before big ask (purchase)
  • Quiz funnels: They answer questions = commitment to process
  • Progress tracking: "You're 60% complete" encourages finishing
  • Email sequences: Each opened email is a micro-commitment

7. Loss Aversion

What it is: People fear losing something more than they desire gaining it.

  • Frames losses: "Don't lose another year struggling with [problem]"
  • Money-back guarantees: Remove purchase risk
  • Show cost of inaction: "Every day without this costs you $X"
  • Example: "Still doing this manually? You're losing 10 hours/week"

8. Anchoring Effect

What it is: First number seen becomes reference point for value.

  • Original price: "$2,997 value" then show discount "$997 today"
  • Tiered pricing: Show expensive option first makes mid-tier seem reasonable
  • Comparison: "Hiring an agency: $50K | Our course: $997"
  • Bundle value: "Everything together worth $5,000, today only $497"
üí° Combine Multiple Principles: "Join 5,000+ members (social proof). Only 10 spots left at this price (scarcity). Price increases Sunday (urgency)." Layering psychology amplifies conversion power.

‚úçÔ∏èCopywriting Formulas That Convert

Stop staring at blank pages. Use these proven formulas to write compelling headlines, emails, and sales pages fast.

1. PAS Formula (Problem-Agitate-Solution)

Best for: Landing pages, emails, sales pages

  • Problem: Identify the pain point
    • "Are you struggling to get quality leads from your website?"
  • Agitate: Make the problem feel urgent
    • "Every day without a proper funnel, you're losing potential customers to competitors. Your website traffic is worthless if it doesn't convert."
  • Solution: Present your offer as the fix
    • "That's why I created [Product] - a proven system that converts 35% of visitors into qualified leads."

2. AIDA Formula (Attention-Interest-Desire-Action)

Best for: Full sales pages, webinar scripts, video sales letters

  • Attention: Grab them with bold headline
    • "How I Generated $100K in 90 Days With One Simple Funnel"
  • Interest: Explain what's in it for them
    • "In this training, I'll show you the exact 7-step system I used, broken down into simple, repeatable steps."
  • Desire: Make them want it (benefits, social proof)
    • "Imagine waking up to notifications of sales while you sleep. That's what happened for 2,000+ students."
  • Action: Tell them exactly what to do
    • "Click the button below to enroll for just $497 (50% off for the next 3 hours)."

3. BAB Formula (Before-After-Bridge)

Best for: Testimonials, case studies, social media posts

  • Before: Where they were (relatable struggle)
    • "6 months ago, I was making $3K/month as a freelancer, working 60-hour weeks."
  • After: Where they are now (dream outcome)
    • "Today, I'm at $15K/month with automated funnels, working just 20 hours/week."
  • Bridge: How they got there (your solution)
    • "The turning point was implementing the funnel system from [Your Course]."

4. FAB Formula (Features-Advantages-Benefits)

Best for: Product descriptions, bullet points, comparisons

  • Features: What it is (technical specs)
  • Advantages: What it does (functionality)
  • Benefits: What it means for them (transformation)
  • Example:
    • Feature: "Drag-and-drop funnel builder"
    • Advantage: "Create pages in minutes without coding"
    • Benefit: "Launch your funnel this weekend instead of waiting months"

5. The 4 U's Formula (Headlines)

Best for: Headlines, subject lines, ad copy

  • Useful: Does it offer clear value?
  • Urgent: Does it create time pressure?
  • Unique: Does it stand out from competitors?
  • Ultra-Specific: Does it include numbers/details?
  • Example: "How to Build Your First $10K/Month Funnel in 30 Days (Even as a Complete Beginner)"
    • ‚úÖ Useful: Learn to build funnel
    • ‚úÖ Urgent: 30 days
    • ‚úÖ Unique: For complete beginners
    • ‚úÖ Ultra-Specific: $10K/month, 30 days

6. The "So What?" Test

Best for: Benefit-driven copy everywhere

  • For every feature you write, ask "So what?"
  • Feature: "Our software has AI automation"
  • So what? "It saves you 10 hours per week on manual tasks"
  • So what? "That's 40 hours per month to work on growth instead of busy work"
  • So what? "Which means you can finally take that vacation you've been postponing"
  • Keep asking until you reach emotional benefit

7. Power Words Library

Replace weak words with powerful alternatives:

  • "Buy" ‚Üí "Invest in, Claim, Secure, Get instant access to"
  • "Free" ‚Üí "Complimentary, Bonus, No-cost, On us"
  • "New" ‚Üí "Fresh, Cutting-edge, Revolutionary, Breakthrough"
  • "Good" ‚Üí "Proven, Battle-tested, Industry-leading, Premium"
  • "Save" ‚Üí "Keep, Protect, Preserve, Safeguard"
  • Use sparingly - don't overload with hyperbole

‚ö†Ô∏è15 Common Funnel Mistakes to Avoid

Learn from others' failures. These mistakes cost businesses thousands in lost revenue.

1. No Clear Value Proposition

  • Mistake: Visitor lands on page and can't tell what you do in 5 seconds
  • Fix: Headline should state: WHO it's for + WHAT they get + OUTCOME
  • Example: "Help B2B SaaS Companies Generate 50+ Qualified Leads Per Month"

2. Too Many Form Fields

  • Mistake: Asking for 10 pieces of info when email is enough
  • Impact: Each extra field reduces conversion by 5-10%
  • Fix: Ask for minimum needed (name + email). Collect more later.

3. Slow Page Load Speed

  • Mistake: Page takes 5+ seconds to load (especially on mobile)
  • Impact: 40% of visitors leave if page takes over 3 seconds
  • Fix: Compress images, use CDN, minimize code, test on mobile first
  • Tools: Google PageSpeed Insights, GTmetrix

4. Weak or Missing Call-to-Action

  • Mistake: CTA says "Submit" or "Click Here" (zero benefit)
  • Fix: Use action + benefit: "Get My Free Template", "Start 14-Day Trial"
  • Also: Make CTA button large, contrasting color, appears 2-3x on page

5. Not Mobile-Optimized

  • Mistake: Funnel looks great on desktop, terrible on phone
  • Impact: 60-70% of traffic is mobile - you're losing majority of visitors
  • Fix: Design mobile-first, test on iPhone and Android, large tap targets

6. No Social Proof

  • Mistake: Zero testimonials, reviews, or trust signals
  • Impact: Visitors don't trust unknown brands without validation
  • Fix: Add 3-5 testimonials with photos, case studies, client logos

7. Complicated Checkout Process

  • Mistake: Multi-step checkout, forced account creation, limited payment options
  • Fix: One-page checkout, guest checkout option, offer PayPal + cards
  • Remove friction: Auto-fill address, show progress indicator

8. No Follow-Up Sequence

  • Mistake: Someone opts in, gets lead magnet, never hears from you again
  • Impact: Leaving 80% of revenue on the table
  • Fix: Create 5-7 email nurture sequence over 2 weeks minimum

9. Ignoring Analytics

  • Mistake: No tracking installed, don't know where visitors come from or drop off
  • Fix: Install GA4, Facebook Pixel, track conversions, review weekly
  • What to track: Traffic sources, bounce rate, conversion rate, drop-off points

10. Generic, Boring Copy

  • Mistake: "We offer quality services at affordable prices" (meaningless)
  • Fix: Be specific, use numbers, focus on transformation not features
  • Example: "Help SaaS founders book 20+ qualified demos per week"

11. No Urgency or Scarcity

  • Mistake: Offer available forever with no reason to act now
  • Impact: Visitors say "I'll come back later" (they never do)
  • Fix: Add deadline, limited spots, or time-sensitive bonus

12. Ignoring Cart Abandonment

  • Mistake: 70% abandon cart, no recovery emails sent
  • Fix: Send 3 abandonment emails over 48 hours
  • Recovery rate: 10-15% of abandoned carts can be recovered

13. No Risk Reversal

  • Mistake: No guarantee, refund policy, or trial period
  • Impact: Risk averse customers won't buy
  • Fix: Offer 30-day money-back guarantee, free trial, or "cancel anytime"

14. Trying to Sell Too Fast

  • Mistake: Cold traffic lands on page ‚Üí immediate $2,000 offer
  • Fix: Warm them up first: Free content ‚Üí Email nurture ‚Üí Then pitch
  • Customer journey: Awareness ‚Üí Interest ‚Üí Desire ‚Üí Action (don't skip steps)

15. Not Testing Anything

  • Mistake: Launch funnel, assume it's optimized, never A/B test
  • Impact: Missing easy 20-50% conversion improvements
  • Fix: Test headlines, CTAs, images monthly. Small lifts compound.

üìñFunnel Metrics Glossary

Speak the language of marketing. Know these terms to communicate professionally with clients.

Traffic & Engagement Metrics

  • Impressions: Number of times your ad/page was shown
  • Reach: Number of unique people who saw your content
  • Clicks: Number of times someone clicked your link/ad
  • CTR (Click-Through Rate): (Clicks √∑ Impressions) √ó 100. Good CTR: 2-5%
  • Bounce Rate: % of visitors who leave immediately. High bounce (60%+) = problem
  • Time on Page: How long visitors stay. Longer = more engaged

Conversion Metrics

  • Conversion Rate: (Conversions √∑ Visitors) √ó 100
  • Lead: Someone who gave you contact info (email, phone)
  • MQL (Marketing Qualified Lead): Lead who matches target criteria
  • SQL (Sales Qualified Lead): Lead ready for sales conversation
  • Opt-in Rate: % of visitors who submit form (20-40% is good)
  • Sales Conversion Rate: % of leads who become customers

Revenue & Cost Metrics

  • AOV (Average Order Value): Total revenue √∑ number of orders
  • LTV (Lifetime Value): Total revenue from customer over entire relationship
  • CAC (Customer Acquisition Cost): Total marketing spend √∑ new customers
  • CPA (Cost Per Acquisition): Same as CAC
  • CPL (Cost Per Lead): Marketing spend √∑ leads generated
  • CPC (Cost Per Click): Ad spend √∑ clicks received
  • CPM (Cost Per Mille): Cost per 1,000 impressions
  • ROAS (Return on Ad Spend): Revenue √∑ ad spend. 3:1 minimum, 5:1+ excellent
  • ROI (Return on Investment): (Revenue - Cost) √∑ Cost √ó 100

Email Marketing Metrics

  • Open Rate: % who opened email. 20-30% is good
  • Click Rate: % who clicked link in email. 2-5% is good
  • Unsubscribe Rate: % who opted out. Under 0.5% is healthy
  • List Growth Rate: New subscribers - unsubscribes √∑ total list
  • Email Deliverability: % of emails that reach inbox (not spam). Aim for 95%+

Funnel-Specific Metrics

  • Funnel Conversion Rate: % who complete entire funnel (visitor ‚Üí customer)
  • Drop-off Rate: % who leave at each funnel stage
  • Average Time to Convert: Days from first visit to purchase
  • Touchpoints to Conversion: Number of interactions before buying (7-13 typical)
  • Cart Abandonment Rate: % who add to cart but don't buy (70% average)
  • Upsell Acceptance Rate: % who accept post-purchase offer (10-30%)

Advanced Performance Metrics

  • EPC (Earnings Per Click): Total earnings √∑ total clicks (affiliate marketing)
  • Churn Rate: % of customers who cancel subscription monthly
  • MRR (Monthly Recurring Revenue): Predictable revenue from subscriptions
  • ARR (Annual Recurring Revenue): MRR √ó 12
  • Payback Period: Months to recover CAC from customer revenue
  • NPS (Net Promoter Score): Customer satisfaction metric (-100 to +100)
üí° The Golden Rule: LTV should be at least 3x your CAC. If it costs you $100 to acquire a customer, they should generate $300+ in lifetime revenue. If not, your funnel economics don't work.

  • Emotions: Trust, security, calm, professionalism, reliability
  • Use for: B2B services, healthcare, finance, technology
  • Industries: Banks, insurance, SaaS, medical, corporate
  • Conversion impact: Builds trust, reduces anxiety about purchase

  • Use for: Environmental products, health, finance, "proceed" actions
  • Industries: Organic/natural products, sustainability, wellness

  • Emotions: Confidence, enthusiasm, friendliness, affordability
  • Use for: CTA buttons, limited offers, fun brands
  • Industries: E-commerce, creative services, food
  • Conversion impact: High visibility without red's aggression

Purple (#800080 spectrum)

  • Emotions: Luxury, creativity, wisdom, spirituality, royalty

Font Pairing Rules

  • Rule 2: Contrast fonts (serif headline + sans-serif body, or vice versa)
  • Rule 3: Keep weights different (bold headline + regular body)
  • Examples of good pairs:

Font Size Guidelines (Desktop)

  • H1 (Main Headline): 36-48px (larger = more impact)
  • H2 (Section Headers): 24-32px
  • H3 (Sub-headers): 20-24px
  • Body Text: 16-18px (never smaller than 16px)
  • CTA Button Text: 18-22px (needs to be readable from distance)
  • Small Print/Disclaimers: 12-14px minimum

Font Size Guidelines (Mobile)

  • H1: 28-36px (smaller than desktop due to screen size)
  • H2: 22-28px
  • Body Text: 16px minimum (iOS requirement for no auto-zoom)
  • Line Height: 1.5-1.6x font size for readability
  • Test: Always view on actual mobile device, not just browser resize

Readability Best Practices

  • Line Length: 50-75 characters per line (optimal for reading)
  • Line Height: 1.5x to 1.8x the font size (more space = easier to read)
  • Paragraph Length: 3-5 sentences maximum (scannable chunks)
  • Text Color: Dark gray (#333333) often easier to read than pure black
  • Letter Spacing: Slight increase (1-2%) improves headline readability
  • ALL CAPS: Use sparingly (harder to read, feels like shouting)

Font Loading Performance

  • Limit fonts: Each font adds load time (2 fonts = sweet spot)
  • Limit weights: Only load weights you'll use (regular + bold typically enough)
  • Font Display: Use font-display: swap to prevent invisible text
  • System fonts: Fastest option (Arial, Helvetica, Georgia already on device)
  • Web fonts: Google Fonts, Adobe Fonts (slower but more variety)
